Query for Creating tbale:
CREATE TABLE REPORT
(QUERY	TEXT	NOT NULL,
RESULT	TEXT	NOT NULL);


Query to Insert the data: 
INSERT INTO REPORT(
	QUERY,RESULT) 
	VALUES('HELLO THERE', 'HEY');


Query to update the data:
UPDATE REPORT
	SET QUERY= 'HOW YOU DOING', RESULT= 'GOOD'
	WHERE QUERY= 'HELLO THERE', RESULT= 'HEY';

Query to read table:
SELECT * FROM REPORT
